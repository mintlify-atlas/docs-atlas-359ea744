Prowl.Vector is a high-performance mathematics library built for the [Prowl Game Engine](https://github.com/ProwlEngine/Prowl) and available as a standalone NuGet package. It provides double-precision (64-bit) and single-precision (32-bit) types for vectors, matrices, and quaternions — with a clean, Unity-inspired API that feels immediately familiar to game and graphics developers.

## Get Started

<Steps>
  <Step title="Install the package">
    Add Prowl.Vector via NuGet to your .NET project:
    ```bash
    dotnet add package Prowl.Vector
    ```
  </Step>
  <Step title="Import the namespace">
    ```csharp
    using Prowl.Vector;
    using Prowl.Vector.Geometry; // for AABB, Intersection, GJK, etc.
    ```
  </Step>
  <Step title="Start computing">
    ```csharp
    var position = new Double3(1.0, 2.0, 3.0);
    var forward  = Double3.Normalize(new Double3(0, 0, -1));
    var transform = Double4x4.CreateTRS(position, Quaternion.Identity, Double3.One);
    ```
  </Step>
  <Step title="Explore the docs">
    Head to [Core Types](/core/vectors) to learn about vectors, matrices, and quaternions, or jump straight to the [API Reference](/api/double3) for complete type signatures.
  </Step>
</Steps>

<Note>
  Prowl.Vector targets **.NET 6, 7, 8, 9, 10** and **netstandard2.1**. It has no runtime dependencies beyond the .NET base class library.
</Note>
